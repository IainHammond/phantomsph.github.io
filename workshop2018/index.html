1st Phantom Users Workshop

19th-23rd Feb 2018, Monash University (Clayton Campus), Melbourne, Australia

1st Phantom Users Workshop

It's been a year since Phantom was released into the wild...

1st Phantom Users Workshop

So it's time to gather the user community together, solve some problems and set some goals

PROGRAM

Program outline:

  • Monday 19/2/2018 — Intro to programming in Fortran (10am - 4pm)
  • Tuesday 20/2/2018 — Introduction to Smoothed Particle Hydrodynamics (10am - 4pm)
  • Wednesday 21/2/2018 — Phantom users workshop (9am-5pm)
  • Thursday 22/2/2018 — Phantom users workshop (9am-5pm)
  • Friday 23/2/2018 — Phantom users workshop (9am-3pm)

Confirmed invited speakers:

Daniel Price (Monash University) — Overview of SPH and Phantom
James Wurster (University of Exeter, UK) — Phantom overview
Terrence Tricco (CITA, University of Toronto, Canada) — Magnetic fields in Phantom
Christophe Pinte (Monash University and CNRS, Grenoble, France) — Radiative transfer with Phantom-MCFOST
